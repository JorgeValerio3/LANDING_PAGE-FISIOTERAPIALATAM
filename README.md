# UFAAL - Unión de Fisioterapia Acuática de América Latina

![UFAAL Logo](./client/src/assets/logo.png)

## 🌊 Sobre el Proyecto

Este repositorio contiene la **Landing Page oficial de UFAAL**, la red líder de profesionales en fisioterapia acuática de Latinoamérica. El sitio web está diseñado para integrar a profesionales, académicos e instituciones comprometidas con el desarrollo y la promoción de la fisioterapia acuática en la región.

### 🎯 Misión
Promover el desarrollo, fortalecimiento y posicionamiento de la fisioterapia acuática en América Latina, impulsando la educación continua, la investigación científica y la práctica profesional ética.

### 👁️ Visión
Ser una red latinoamericana de referencia internacional en fisioterapia acuática, reconocida por impulsar la formación especializada, la investigación científica y la práctica clínica basada en evidencia.

## 🛠️ Stack Tecnológico

El proyecto está desarrollado utilizando herramientas modernas para garantizar rendimiento y escalabilidad:

- **Framework:** [React 18](https://reactjs.org/)
- **Build Tool:** [Vite](https://vitejs.dev/)
- **Lenguaje:** [TypeScript](https://www.typescriptlang.org/)

- **Animaciones:** [Framer Motion](https://www.framer.com/motion/)

## 🚀 Comenzando

### Requisitos Previos
- Node.js (v18 o superior)
- npm o yarn

### Instalación

1. Clona el repositorio:
   ```bash
   git clone <url-del-repositorio>
   ```

2. Entra al directorio del cliente:
   ```bash
   cd client
   ```

3. Instala las dependencias:
   ```bash
   npm install
   ```

### Desarrollo

Para iniciar el servidor de desarrollo:
```bash
npm run dev
```

### Construcción para Producción

Para generar los archivos estáticos:
```bash
npm run build
```
Los archivos se generarán en la carpeta `dist`.

## 📁 Estructura del Proyecto

```text
/client
  /src
    /components   # Componentes reutilizables y secciones (Hero, QuienesSomos, etc.)
    /data         # Archivos de contenido dinámico (content.json)
    /assets       # Imágenes y recursos estáticos
    /services     # Lógica de servicios y APIs
  /public         # Archivos públicos
/docs             # Archivos para despliegue en GitHub Pages
/docs_proyecto    # Documentación institucional (Estatutos, Historia, etc.)
```

## 🌐 Despliegue

El sitio está configurado para desplegarse automáticamente en **GitHub Pages**. Los archivos estáticos se sirven desde la carpeta `docs` en la raíz del proyecto.
